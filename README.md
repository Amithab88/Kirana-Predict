# Kirana-Predict: Smart Inventory Forecaster
An AI-lite tool designed for Indian Kirana stores to prevent stock-outs and reduce waste.

## 🚀 Features
- **Sales Analysis:** Identifies top-selling products and weekly trends.
- **Inventory Prediction:** Calculates "Burn Rate" and predicts stock depletion dates.
- **Modular Design:** Built with a clean Python structure for scalability.

## 🛠️ Tech Stack
- Python 3.12
- Pandas (Data Processing)
- Modular Architecture

## 📖 How to Run
1. Ensure all the requirements are installed: `pip install -r requirements.txt`
2. Place your sales data in `data/grocery_chain_data.csv`
3. Run the main app: `python main.py`
4. Run the app.py : 'python -m streamlit run app.py'
